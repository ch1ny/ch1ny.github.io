
![预览图](cover.png)

## ✨ 特性
 - 图片懒加载
 - 内置多语言
 - 支持多款评论插件
 - 网页访问统计
 - 侧栏备案信息
 - 灯箱大图
 - 图片瀑布流
 - 社交按钮

## 🎓 使用说明
请查看 [https://docs.nexmoe.com/](https://docs.nexmoe.com/)
