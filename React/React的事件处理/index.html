React 中的事件处理

React 元素的事件处理和 DOM 元素的很相似，但是有一点语法上的不同：

  • React 事件命名采用驼峰写法（camelCase），而不是全部小写；
  • 使用 JSX 语法时需要传入一个函数作为事件处理函数，而非字符串。
